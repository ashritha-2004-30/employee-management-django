# Employee Management System (Django)

This is a web-based application built using Django to manage employee data.

## Features
- Add Employee
- Edit Employee
- Delete Employee
- View Employee List

## Technologies Used
- Python
- Django
- HTML
- CSS

## How to Run
1. Install Django
2. Run: python manage.py runserver
3. Open: http://127.0.0.1:8000/
